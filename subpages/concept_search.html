Concept Search

An iterative search algorithm that uses natural language processing to provide a larger corpus of relevant results.

Overview

Implemented in Python. Extracts relevant and irrelevant keywords during each search to refine results. Uses a tree structure to group results by iteration.

Resources

  • GloVe - word-vector representations
  • IBM Watson Natural Language Understanding, TF-IDF - keyword extraction
  • Gensim - latent dirichlet allocation for topic modeling
  • NLTK - text pre-processing

Research Partners

  • Text extraction algorithm - Mateus Winkelmann, Hannah Braun

Control Flow

Input query

Locate context document

Query google, obtain search results

Extract text from results

Compare text to context document

Use most and least relevant documents to create new queries
